# 42-C-fractal
# envionment
mac os x el capitan

mini lib x

langage C

# Description 
This programm draw different fractals : Mandelbrot, Julia, burning ship, bird of prey, etc...

# run :
git clone https://github.com/rim31/42_C_fratol.git

cd 42-C-fractal

make

./fractol 1

(try other fractal with a number between 1 - 8)
